  먼저 XAML의 경우, 아래와 같이 라이브러리를 설치해줍니다.

2. .cs 파일

  일반적으로 Ctrl + K, Ctrl + D를 하면 정렬을 할 수 있습니다.
  그러나 이렇게 하면 작업 중 몇몇 코드 정렬이 누락될 수 있습니다.
  VS Code는 format on save 기능을 사용하여 저장할 때마다 코드를 자동으로 정렬할 수 있습니다.

  ![image](./Images/image02.png)
